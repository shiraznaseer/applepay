Base URLs

Local: http://localhost:<port>
Production example: https://applepay.tamarran.com

Authentication / Headers

Tabby webhook validation: expects header X-Tabby-Signature (basic comparison to configured secret in this project).
Tamara webhook validation: expects JWT token in either:
  • ?tamaraToken=<jwt> query string, or
  • Authorization: Bearer <jwt>

POST /api/tamara/webhook
Important: This endpoint requires a valid Tamara notification JWT. It also:
  • Logs to Logs/tamara-webhook.log
  • Saves history to SQL if Tamara:DbConnectionString is configured
  • Broadcasts WebSocket event tamara.order.updated
  • On order_approved, it performs authorise then capture server-to-server.

Request headers (option 1)

Authorization: Bearer <tamara_notification_jwt>
Content-Type: application/json

Request (option 2)

POST /api/tamara/webhook?tamaraToken=<tamara_notification_jwt>

Request body (example)

{
  "event_type": "order_approved",
  "order_id": "TAMARA-ORDER-ID",
  "status": "approved"
}

Response

{
  "received": true
}
